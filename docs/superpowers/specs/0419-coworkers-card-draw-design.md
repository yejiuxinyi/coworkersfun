## 1. 产品一句话

借势 SBTI 的社交传播模式，做一款**职场版"抽老板/同事卡"**的H5娱乐应用：用户用 5 道**玄学小问题**（星座/昨天股票亏没亏/有没有人表白/吃没吃到好吃的…）触发一张专属的"今日出战领导/同事"卡牌，看到角色名 + 配图 + 经典吐槽（或暖心）台词，一键分享裂变朋友圈。

**核心情绪**：每天打开，带点运势占卜感地告诉你——今天你要面对谁。

## 2. 灵感来源与差异

| 参考 | 他有什么 | 我们怎么变 |
|---|---|---|
| SBTI（B站蛆肉儿串儿） | 31题 → 27种人格 | 5题 → 48张卡，缩短链路 |
| MBTI | 纯人格测评 | 换成"职场角色 + 台词梗" |
| 原神/崩铁抽卡 | 稀有度金光动画 | 复用视觉语言，降低学习成本 |

**核心差异点**：SBTI 是自我投射（"我是谁"），我们是**对象投射**（"我今天要面对谁"）。解决的情绪是"职场吐槽欲 + 同事共鸣"。

### 5 道玄学快问快答（示例）

**设计原则**：问题本身是**玄学向、非职场直指**的，制造"占卜感"而非"测评感"。用户答题时不该能预判自己会抽到啥。

1. 你的星座？（12星座选1，后端按四象限映射—水象/火象/土象/风象）
2. 昨天股票/基金状态？（A 亏麻了 / B 微赚 / C 没买是唯一赢家 / D 装死没看）
3. 最近一周有没有人跟你表白？（A 有 / B 没有 / C 我跟别人表白了 / D 不想谈）
4. 昨天有没有吃到好吃的？（A 吃到了 / B 勉强吃饱 / C 忘了 / D 减肥中）
5. 今天心情量化？（A 想辞职 / B 麻了 / C 还行 / D 今天是好日子）

**答题与抽卡的关系**（随机为主，逻辑为辅）：
- 每道题选项都带一个 `luck` 值（-2 ~ +2）和可选 `tag`（如 `water_sign`、`loved`、`lucky`）
- 总 `luck_score = Σ luck` → 影响稀有度概率池的微调
- `tags` 集合 → 影响"命定路径"判定

```
基础概率池: R 75% / SR 20% / SSR 5%
luck_score >= +6 → R 60% / SR 30% / SSR 10%（好运加成）
luck_score <= -4 → R 85% / SR 13% / SSR 2%（黑运打折）
其他 → 基础池
```

## 5. 卡池设计（48张）

### 稀有度分布

| 稀有度 | 数量 | 抽中概率（基准） | 类型 | 视觉 |
|---|---|---|---|---|
| R  普通 | 36 张 | 75% | 值得吐槽型 | 灰/白色边框 |
| SR 稀有 | 8 张  | 20% | 中等精品/偶尔暖心 | 银色+轻光效 |
| SSR 传说 | 4 张 | 5%  | 天使领导（梦中的） | 金色+粒子特效 |

吐槽型（R+部分SR） : 精品型（部分SR+SSR）≈ 3 : 1（与用户要求对齐）

### R 层 36 张（普通吐槽型）

基于用户原稿 34 个 + 新增 2 个：
> 考古学家、闲鱼买家、健身教练、生物学家、天师、魔方、拖把、时间旅行者、篮球裁判、长跑裁判、收藏家、初恋守护者、失忆症患者、王阳明、美食家、美国人、文盲、苹果树、马克思、科学家、碰碰车、阿西莫夫、慈善家、温度计、海王、机器人反抗军、物理学家、恐怖分子、狙击手、亚里士多德、宝可梦训练师、烘干机、东亚父亲、拉面师傅 + **变色龙**（对老板一套对下属一套）+ **透明人**（开会从不表态甩锅第一个）

### SR 层 8 张（候选，需精修台词）

1. **树洞** — "最近有啥困难跟我说"（真的会听）
2. **哨兵** — 帮你挡掉无效会议
3. **翻译官** — 能把老板的"你看着办"翻译成人话
4. **面包师** — 加班真的请你吃饭
5. **沙漠绿洲** — 别的组卷，他的组准点下班
6. **教练** — 手把手带你成长
7. **保护伞** — 跨部门 battle 永远站你这边
8. **阳光型人格** — 每天见你第一句是"早，今天状态不错"

### SSR 层 4 张（候选）

1. **诸葛亮** — "我来写，你休息"
2. **盖茨** — 年会发万元红包 / 股票池直接拉你进
3. **甘道夫** — "你已经做得很好了"（真诚不敷衍）
4. **圣诞老人** ⭐ — 年底真的涨薪 + 发年终（**命定卡，见 §5.5**）

> 说明：SR/SSR 台词/配图在 plan 阶段再精修，此处仅框架。

### 5.5 命定路径（SSR的"隐藏关"）

用户反馈："最好的 SSR 只有一种路径 + 需要一些运气才能出。"

**设计**：`SSR001 圣诞老人` 是**命定卡**，必须同时满足：
1. **路径条件**（tags 命中）：`fire_sign` + `lucky` + `loved` + `foodie` + `good_day`（约等于"星座是火象 + 股票赚 + 被表白 + 吃到好吃的 + 今天好日子"全中）
2. **运气条件**：通过一次 5% 独立 roll

只有路径+运气双通过才掉圣诞老人；否则哪怕 luck_score 爆表，也只在 `{诸葛亮/盖茨/甘道夫}` 三张 SSR 池里抽。

伪代码：
```
if tags ⊇ {fire_sign, lucky, loved, foodie, good_day} and random() < 0.05:
    return "SSR001_圣诞老人"
else:
    走正常概率池（可能抽到其余 3 张 SSR）
```

其他 3 张 SSR 走正常稀有度概率池，与 R/SR 混抽。

## 7. 配图方案（AI 批量生成）

- **风格**：扁平矢量插画 + Q版头像 + 统一色卡（紫/粉/蓝 3色调对应 R/SR/SSR）
- **工具**：优先 Gemini `imagen-3` 或 Midjourney（一致性 prompt），48张一次性生成
- **规格**：768×1024 竖版（卡牌比例），webp 压缩
- **产出目录**：`assets/cards/{id}.webp`
- **prompt 模板**（示例）：
  > flat vector illustration, Q-version chibi character, [ROLE_DESCRIPTION], workplace setting, [RARITY_COLOR] background, trading card art, no text

## 11. 风险与对齐点

- **版权**：角色名/台词里如"王阳明/马克思/亚里士多德/甘道夫"等公众知名名称仅作梗用，描述保持玩梗界限；必要时用"诸葛亮→诸葛式老板"这种软化
- **冷启动**：48张一次性画完成本不小，可先用 emoji + 描边框做占位（第一版），AI 图作为 v1.1
- **加权玄学**：用户答题其实希望"随机有惊喜"，不宜搞得太精确，bias 权重保持"微弱影响"即可
