## 🎨 卡面角色风格（4 选 1 或自提）

标准卡通小人 + 头顶不同道具（温度计 / 哨子 / 算盘 / 拖把 ...）。

### 方案 A · 2 头身扁平矢量 Q 版（Notion 风 / Alegria）
- 头大身小，扁平色块，最多 2 级阴影
- 黑描边粗 2-3px
- 风格关键词：`flat vector illustration, chubby 2-head-proportion, thick black outline, saturated pastel colors, hand-drawn feel`
- **优点**：现代、可爱、量产一致性好
- **缺点**：略"AI 插画味"
- **参考效果**：Notion 官网插画

### 方案 B · 3 头身手绘线稿 + 淡彩（Duolingo 风）
- 比例更自然，线稿粗细有节奏
- 水彩或油画笔触上色
- 风格关键词：`hand-drawn character, 3-head-proportion, rough ink outline, soft watercolor fill, playful mascot style`
- **优点**：温暖、有人情味，像一本漫画书
- **缺点**：量产一致性稍差，需要精修
- **参考效果**：Duolingo 猫头鹰、Mailchimp 插画

### 方案 C · 棍人 meme（极简火柴人）
- 只画头 + 简单四肢线条
- 头顶夸张道具反差
- 风格关键词：`minimalist stick figure, exaggerated prop on head, single black line, white background, xkcd style`
- **优点**：最有梗，复刻 xkcd / 表情包气质，容易病毒传播
- **缺点**：显得"敷衍"，长期看审美疲劳
- **参考效果**：xkcd 漫画、"张三李四"火柴人 meme

### 方案 D · 像素 8-bit（复古游戏风）
- 16x16 或 32x32 像素头像
- 头上道具用 8-bit 图标
- 风格关键词：`8-bit pixel art character, 32x32 sprite, retro gameboy palette, chunky outlined`
- **优点**：游戏感强，和"抽卡"机制契合
- **缺点**：表情和细节受限，不够情绪化

## 🐶 Doge 卡背风格（4 选 1 或自提）

5 张盲盒的卡背，全部同一个。

### 方案 1 · 经典 Doge meme 原图
- 侵权风险低（公共梗），但画质粗糙
- 搭配"such work much job"一类英文梗
- 适合：想做"纯整活"风格

### 方案 2 · 重绘 Q 版柴犬
- 精致、配色可控
- 失去梗味，像普通宠物插画
- 适合：想走"精品盲盒"调性

### 方案 3 · 极简几何狗头（**当前 demo 就是这个**）
- 圆脸 + 三角耳朵 + 两点眼睛
- 柠檬黄底 + 黑描边
- 强调识别度，弱化细节
- 适合：和赛博卡通整体统一

### 方案 4 · 烫金/压印盲盒质感
- 黑卡金线稿，边框立体
- 像真实盲盒封面
- 适合：想强化"拆盒仪式感"
- 实现难度高（需要复杂阴影 / 真实材质）

## 🤔 我的组合推荐

| 你的诉求 | 我推 | 理由 |
|---|---|---|
| 想做**传播裂变**型，朋友一看就发 | **C 棍人 + 1 meme Doge** | 最有梗，复刻表情包气质 |
| 想做**品质盲盒**型，可长期做 | **A 扁平 + 4 烫金** | 有质感有统一性 |
| 中间路线（和当前赛博卡通色匹配最好） | **A 扁平 + 3 几何狗头** | demo 无缝升级，改动最小 |
| 想游戏化 | **D 像素 + 3 几何** | 整体像一个街机游戏 |

**我本能偏向：A + 3** — 当前 demo 改动最小，视觉自洽，适合你慢慢迭代。
